Weather Dashboard

This was a great learning experience and the perfect opportunity to experiment with various different coding concepts. I built the styling using uikit framework. I also created an object with methods to help dynamically create html elements to append to the DOM using jquery. I used Open Weather API to populate the current weather and forecasting data. jQuery was used to make the API calls. Although uikit is a great framework and has excellent documentation, it was a challenge working with nested HTML elements and jQuery to dynamically manipulate the DOM; however, it was extremely rewarding once I got it working!

Jedi University

This was my first group project and collaborative effort to bring forth this Jedi University. The idea is to satisfy your inner Star Wars nerd with some stats on Planets, People, Ships, Species and Films. A learning experience this was, to work as a team from conception, to divvying out tasks, resolving merge conflicts, approving pull requests and presenting our finished product to 50+ people. Github was one of the most challenging pieces of the project as we had to learn all the various commands from creating branches, to pushing and pulling. We used jQuery for our API calls to the Star Wars API (SWAPI) and GIPHY to bring you all the Star Wars content. We also used jQuery for DOM manipulation. We dipped into using uikit for mobile first design and the overall look and feel of the website.
